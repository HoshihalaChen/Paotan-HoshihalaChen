  跑团助手 — 免费部署指南（国内可直接访问）
══════════════════════════════════════════════════════════

推荐方案：Gitee Pages（码云）— 国内最稳定
备选方案：GitHub Pages + Vercel — 国际通用
══════════════════════════════════════════════════════════


▎方案一：Gitee Pages（推荐 ★★★★★）

  优点：纯国内平台，速度快，无需VPN
  缺点：需要实名认证

  步骤：

  1. 注册 Gitee 账号
     打开 https://gitee.com/ → 注册 → 完成实名认证

  2. 创建仓库
     - 登录后点击右上角「+」→「新建仓库」
     - 仓库名称填：paotuan-app
     - 选择「公开」
     - 勾选「使用 Readme 文件初始化仓库」
     - 点击「创建」

  3. 上传文件
     - 进入仓库 → 点击「上传文件」
     - 把 dist 文件夹内的全部文件拖拽上传：
       index.html
       index-*.css
       index-*.js × 2
     - 提交信息填「部署」
     - 点击「提交」

  4. 开启 Pages 服务
     - 进入仓库 →「服务」→「Gitee Pages」
     - 部署分支：master
     - 部署目录：不填（根目录）
     - 点击「启动」
     - 等待几分钟，生成访问链接

  5. 获取链接
     格式：https://你的用户名.gitee.io/paotuan-app
     把这个链接发给朋友即可


▎方案二：GitHub Pages + Vercel（推荐 ★★★★）

  优点：免费，部署快，国际通用
  缺点：偶尔需要 VPN 访问

  步骤：

  1. 注册 GitHub 账号
     打开 https://github.com/ → Sign up

  2. 创建仓库并上传
     - 新建仓库 paotuan-app（Public）
     - 把 dist 文件夹内全部文件上传
     - 提交

  3. 部署到 Vercel
     - 打开 https://vercel.com/
     - 点击「Sign Up」→「Continue with GitHub」
     - 授权后 →「New Project」
     - 选择 paotuan-app 仓库
     - 直接点「Deploy」
     - 等待完成，获取链接：https://paotuan-app.vercel.app


▎方案三：Cloudflare Pages（推荐 ★★★★）

  优点：CDN 全球加速，部分节点国内可访问
  缺点：注册需要邮箱

  步骤：

  1. 注册 https://dash.cloudflare.com/sign-up
  2. 左侧菜单 →「Workers & Pages」
  3. 「Pages」→「Upload assets」
  4. 创建项目名，把 dist 文件夹拖入上传区域
  5. 点击「Deploy」
  6. 获取链接：https://你的项目名.pages.dev


▎方案四：surge.sh（最快，5 秒部署 ★★★）

  优点：命令行一键部署，无需注册仓库
  缺点：免费版链接随机

  步骤：
  1. 在 dist 文件夹打开命令行
  2. 运行：npx surge
  3. 首次使用输入邮箱和密码（自动注册）
  4. 回车确认项目路径
  5. 自定义域名或随机域名
  6. 获取链接发给朋友


══════════════════════════════════════════════════════════

  推荐优先级：
  国内朋友 → Gitee Pages（方案一）
  国外朋友 → Vercel（方案二）
  最快部署 → surge.sh（方案四）

  所有方案均免费，无需安装额外软件。
