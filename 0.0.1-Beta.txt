【已实现功能清单】

一、核心架构
  - Vue 3 Composition API（<script setup>）组件化架构
  - Vite 5 构建工具 + HMR 热更新
  - Tailwind CSS 复古羊皮纸风格 UI 系统
  - Pinia 跨页面状态管理（7 个 Store）
  - Dexie.js IndexedDB 本地持久化（8 张数据表）
  - Chart.js 数据可视化（柱状图/雷达图/时间线）

二、AI DM 系统
  - DeepSeek V4 PRO API 流式 SSE 打字机回复
  - 记忆机制：角色卡 + 世界观 + 滑动窗口（30条）防幻觉
  - 主动引导型 DM：强制问句结尾 + 2-3 编号选项 + 沉浸式叙述
  - 模组隔离（PIN）：AI 锁定在选定模组内，禁止串用内容
  - 统一检定系统：属性 + 职业特长 + 装备 + 难度自动加权计算
  - .yes/.no 检定指令：自动计算加成 vs 固定低值

三、7 个功能页面
  1. 主页（HomePage）
     - 8 个本地模组预览卡片 + 系统筛选
     - 模组初始化弹窗（AI 开场叙述 + 角色选择 + 开始冒险）
     - 自定义冒险创建表单 + 队伍头像列表
     - 游戏激活状态横幅 + 锁定机制 + 测试用重置接口

  2. 游戏页（GamePage）
     - 左侧聊天窗口（消息收发 + AI 流式输出 + 选项按钮）
     - 右侧 d20 骰子面板 + 属性快捷检定栏
     - 选项按钮（ChoiceButtons）：解析 AI 编号选项并渲染可点击按钮
     - 战斗面板（CombatStats）：角色/敌人 HP + 属性实时可视化
     - 数据统计面板：投骰柱状图 + 属性雷达图 + 流程时间线
     - 导出对话框 + 日期显示 + 事件计数 + 滚底按钮
     - 游戏结束/主页返回 + 重置确认

  3. 角色页（CharacterPage）
     - 角色列表卡片 + 头像上传接口
     - 六维属性进度条 + HP/MP 进度条
     - 装备清单 + 技能列表 + NPC 好感度进度条
     - 创建/编辑角色弹窗（模组动态选项）

  4. 地图页（MapPage）
     - 地图生成器接口预留（网格背景占位）
     - 只读游戏日期（年/月/日/季节/天数/事件）
     - 地图图层勾选（地形/聚落/路线/地下城/兴趣点）

  5. 世界观页（WorldPage）
     - 模组概览卡片（背景+舞台）
     - 全分类拆分展示（力量/势力体系默认展开+左彩条加强）
     - 左侧快捷导航 + 分类折叠/展开
     - CRUD 条目管理 + 标签系统
     - 模组切换联动

  6. 日志页（LogPage）
     - 左侧任务条目列表 + 右侧正文编辑区
     - 关联人物 + 标签区域
     - CRUD 日志管理

  7. 设置页（SettingsPage）
     - DeepSeek API Key 配置
     - 数据统计面板（会话/角色/消息/备份）
     - 备份管理（列表/下载/删除/清空）
     - 数据导出/清空 + 关于信息

四、8 个本地跑团模组
  D&D 5E × 6：
    - 凡达林的失落矿坑（1-5级）
    - 施特拉德的诅咒（1-10级）
    - 龙枪编年史·龙后之影（1-11级）
    - 湮灭之墓（1-11级）
    - 冰风谷·冰霜少女的雾凇（1-12级）
    - 深水城·龙金劫（1-5级）
  诡秘之主 × 1：
    - 22 条神之途径完整数据（每途径 9 序列+1真神）
    - 序列9魔药选择自动推导神之途径
    - 5 大隐秘组织 + 10 条世界观设定
  克苏鲁的呼唤 × 1：
    - COC 7th 规则 + SAN 值系统 + 1920s 新英格兰设定

  每个模组包含：NPC列表、地点列表、任务列表、物品/装备、预置角色、世界观设定、角色创建选项

五、游戏系统
  - 天数周期：随机起始日期 + 每天 4 事件上限 + 自动推进 + 睡眠触发
  - 角色按模组/会话隔离（sessionId 绑定）
  - 骰子指令：d4/d6/d8/d10/d12/d20/d100 + 自定义表达式 + 优势/劣势
  - 统一检定：属性调整值 + 熟练加值 + 职业特长 + 装备加成 + 难度 DC
  - 拒绝检定（.no）：固定裸 d20 = 7-9

六、数据管理
  - IndexedDB 8 表：sessions/characters/messages/worldInfo/logEntries/diceLogs/mapData/backups
  - 自动备份：每 ≥2 事件 + ≥5 分钟 → 纯文本存档到 DB
  - 备份文件命名：模组名-角色名-D{天数}.txt
  - 备份不受普通重置影响，仅可从设置页手动删除
  - 聊天导出：Markdown / JSON 格式
  - 角色卡导出：JSON 格式

七、UI/UX
  - 低饱和浅奶油复古羊皮纸配色（#F5F0E8 主底 + #3A3A3C 深灰侧边栏）
  - 固定左侧 220px 垂直导航栏 + 星芒罗盘 logo
  - 7 项导航菜单：主页/游戏/角色/地图/世界观/日志/设置
  - 圆角矩形卡片 + 细线阴影 + 羊皮纸 CSS 噪点纹理
  - 全局错误边界（红色横幅）+ 按钮加载状态反馈
  - 自定义滚动条 + 淡入动画 + 打字机光标
