一、如何启动（3 选 1，推荐方法 1）

  ▎方法 1：双击启动脚本（最简单）
    1. 确保电脑已安装 Python（绝大多数 Windows 电脑自带）
       如果没有：https://www.python.org/downloads/
       下载后安装时务必勾选 "Add Python to PATH"
    2. 双击 "启动.bat"
    3. 浏览器会自动打开，没有则手动访问 http://localhost:8080

  ▎方法 2：使用 Node.js
    1. 安装 Node.js：https://nodejs.org/（选择 LTS 版本）
    2. 在此文件夹空白处按住 Shift + 右键 → "在此处打开 PowerShell"
    3. 输入：npx serve -l 8080
    4. 浏览器访问 http://localhost:8080

  ▎方法 3：手动启动 Python 服务器
    1. 在此文件夹空白处按住 Shift + 右键 → "在此处打开 PowerShell"
    2. 输入：python -m http.server 8080
    3. 浏览器访问 http://localhost:8080

  ⚠ 不能直接双击 index.html 文件打开！必须通过上述方式启动。

二、首次使用

  1. 启动后看到主页，左侧是导航栏，右侧是内容区。
  2. 主页展示 8 个可选模组卡片（凡达林/施特拉德/龙枪/诡秘之主/克苏鲁等）。
  3. 点击任意模组的「使用模组 →」按钮，弹出初始化窗口。
  4. AI 会自动生成冒险开场叙述（打字机效果实时展示）。
  5. 在角色选择阶段，选择预设角色或点击「+ 创建新角色」。
     - 点击 🎲 随机生成 可快速创建角色
     - 属性值由职业和种族自动决定，无需手动调整
  6. 点击「开始冒险！」进入游戏。

  ⚠ AI 对话功能已内置 API Key，开箱即用。如需更换自己的 Key：
     左侧导航 → 设置 → 输入新的 API Key → 保存。
     获取 Key：https://platform.deepseek.com/

三、各页面介绍

  ▎主页
    - 顶部横幅显示当前游戏状态
    - 模组卡片：点击「使用模组 →」开始新游戏
    - 自定义冒险：手动输入名称和描述创建
    - 游戏激活后其他模组被锁定，需要「重置游戏」

  ▎游戏页（核心页面）
    左侧：聊天区域
      - 输入消息与 AI 主持人对话
      - AI 回复结尾提供编号选项，点击按钮快速选择
      - 也可自由输入自定义内容

    右侧：骰子面板
      - 投掷 d20 / 自定义骰子
      - 快捷属性检定（选择角色后点击属性名）
      - AI 要求检定时：
        输入 .yes → 自动计算属性+职业+装备加成后投骰
        输入 .no → 放弃检定，获得低于平均的结果
      - 骰子指令：
        .d20         基础 d20 投掷
        .2d6         两个 d6
        .3d8+2       三个 d8 加 2
        .adv         d20 优势（取高）
        .dis         d20 劣势（取低）

    顶部工具栏：
      - ⚔️ 战斗面板：显示角色/敌人 HP 状态
      - 数据统计：投骰柱状图 + 属性雷达图
      - 导出记录：保存聊天为 Markdown

  ▎角色页
    - 查看/创建/编辑角色
    - 上传自定义头像
    - 六维属性、HP/MP 进度条
    - 装备清单、技能、NPC 好感度

  ▎世界观页
    - 模组背景概览
    - 按分类浏览世界设定（力量体系、势力组织、历史、地理等）
    - 点击分类标题展开/折叠
    - 可手动添加/编辑条目

  ▎地图页
    - 游戏日期（系统自动管理，不可手动更改）
    - 地图图层勾选
    - 地图生成器接口预留

  ▎日志页
    - 记录冒险笔记
    - 关联 NPC 和标签

  ▎存档页
    - 模组 → 角色 → 天数 三级存档树
    - 读取存档恢复游戏进度
    - 备份管理：下载/删除纯文本备份

  ▎设置页
    - 配置 AI API Key
    - 自定义每个职业/种族的初始属性值
    - 数据导出/清空

四、常见问题

  Q: 网页打不开 / 显示"无法连接"？
  A: 确认启动脚本正在运行（不要关闭黑色窗口）。
     检查浏览器地址栏是否为 http://localhost:8080。

  Q: AI 不回复 / 回复错误？
  A: 设置页检查 API Key 是否正确。
     本应用内置默认 Key，通常无需额外配置。

  Q: 骰子指令输入后没反应？
  A: 骰子指令必须以 . 开头（如 .d20）。不要省略点号。

  Q: 角色属性怎么是固定的？
  A: 属性由职业+种族自动决定，总和固定为 20 点。
     如需调整：设置页 → 初始属性自定义 → 滑块修改。

  Q: 怎么回到之前的游戏进度？
  A: 左侧导航 → 存档 → 找到对应存档 → 读取。
     游戏过程中天数+1 时自动存档。

  Q: 页面显示异常？
  A: 按 Ctrl+Shift+R 强制刷新。
     关闭浏览器，重新启动服务器，再打开。

  Q: 如何结束游戏换模组？
  A: 游戏页右上角「结束」→ 确认。
     或主页「重置游戏」按钮。

  Q: 端口 8080 被占用？
  A: 把启动命令中的 8080 改成其他数字，如 3000：
     python -m http.server 3000
     然后访问 http://localhost:3000
